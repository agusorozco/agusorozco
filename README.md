# Agus Orozco

Football moves fast, but the interesting stuff lives underneath it — the behaviors, the patterns, the systems that shape how the game actually works at every level.

I'm a builder and analyst working at that layer. Some of what I do lives in data and code; some of it lives in writing. All of it starts with the same question: what's really going on here, and can we build something that reflects that more honestly?

Currently building. Always watching.
